# Политика обработки данных

**Сервис:** Telegram-бот «Хватит»
**Версия:** 1.0 от 10.07.2026

## 1. Какие данные обрабатываются

- **Идентификатор Telegram** (числовой user_id) — для работы аккаунта и синхронизации между устройствами.
- **Номер телефона** — только если вы делитесь контактом в режиме закрытого тестирования; хранится в защищённом виде (хеш и последние 4 цифры), в открытом виде не сохраняется.
- **Финансовые записи, которые вы вносите сами:** суммы, категории, описания операций, балансы счетов, лимиты, настройки.
- **Изображения чеков и выписок**, которые вы отправляете для распознавания, и распознанные из них данные (магазин, позиции, суммы, фискальные реквизиты чека).
- **Технические события** использования Бота (какие функции применялись, без содержимого операций) — для улучшения сервиса.

Бот не запрашивает и не имеет доступа к вашим банковским счетам, картам и криптокошелькам. Все финансовые данные появляются в Боте только потому, что вы ввели их сами.

## 2. Для чего

Данные используются исключительно для работы функций Бота: расчёта ранвея и статистики, распознавания чеков, уведомлений, подписки PRO. Данные **не продаются и не передаются третьим лицам для рекламы**.

## 3. Кому передаются данные

Для работы отдельных функций данные передаются техническим подрядчикам ровно в объёме, необходимом для функции:

| Что передаётся | Кому | Зачем |
|---|---|---|
| Изображение чека/выписки | OpenAI (сервис распознавания) | Распознавание текста и позиций |
| Фискальные реквизиты чека из QR-кода | proverkacheka.com | Получение официальных данных чека |
| Запрос курса валют (без ваших данных) | CoinGecko | Курсы USDT/TON |
| Данные платежа подписки | Telegram | Проведение оплаты Stars |

## 4. Где и сколько хранится

Данные хранятся в базе данных на сервере владельца Бота и не публикуются. Резервные копии создаются автоматически и хранятся ограниченный срок. Данные хранятся, пока вы пользуетесь Ботом.

## 5. Удаление данных

Вы можете в любой момент запросить полное удаление своих данных: отправьте команду **/support** в Боте с текстом «прошу удалить мои данные». Если Бот недоступен — создайте issue в этом репозитории, указав ваш Telegram-username или id. Данные удаляются из базы и перестают попадать в новые резервные копии; старые копии удаляются по истечении срока их ротации.

## 6. Изменения

Актуальная версия Политики публикуется по этому адресу. О существенных изменениях Бот уведомляет сообщением.

## 7. Контакты

Вопросы по данным: команда **/support** в Боте; если Бот недоступен — issue в этом репозитории.
